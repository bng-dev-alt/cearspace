# ClearSpace Design Bible v1.0

> Master design specification for the complete visual redesign of
> ClearSpace.

## Purpose

Redesign the **entire application** while preserving **100%** of: - UX -
information architecture - navigation - workflows - functionality - drag
& drop - interactions

This is a visual redesign only.

------------------------------------------------------------------------

# Design Philosophy

Create a unique design language called **Organic Glass Workspace**.

Keywords:

-   Calm
-   Premium
-   Elegant
-   Editorial
-   Spacious
-   Crafted
-   Modern
-   Timeless

Inspired by Apple, Linear, Notion, Raycast and Framer, but never copying
them.

------------------------------------------------------------------------

# Golden Rule

Every screen must look like it belongs to the same product.

Nothing should feel like a separate admin panel or template.

------------------------------------------------------------------------

# Login Page

The login page defines the emotional first impression.

## Left Panel

-   Large editorial headline
-   Soft layered organic background
-   Ambient lighting
-   Calm gradients
-   Large whitespace
-   Premium typography
-   Brand storytelling

## Right Panel

Floating authentication card.

Contains:

-   Welcome
-   Email
-   Password
-   CTA
-   Social login
-   Register

Use:

-   subtle glass
-   premium shadow
-   rounded corners
-   emerald accent
-   perfect readability

Never place decorative objects over future UI space.

------------------------------------------------------------------------

# Global Design System

## Backgrounds

Use:

-   organic abstract layers
-   soft gradients
-   depth
-   ambient light
-   subtle texture

Never use busy illustrations.

## Glass

Opacity: 90--95%

Very subtle blur.

Thin highlight border.

Layered shadows.

## Radius

Buttons: 14px

Cards: 18px

Dialogs: 24px

Hero cards: 28px

## Typography

Heading: Premium editorial serif.

Body: Inter / Geist / SF Pro style.

## Motion

150--250ms

Ease-out

Small lift

Soft glow

Never flashy.

# Board

Keep the existing Kanban UX exactly.

Improve:

-   Hero
-   Toolbar
-   Statistics
-   Board surface
-   Columns
-   Cards
-   Hover
-   Typography
-   Shadows
-   Spacing

The board should feel like a premium creative workspace.

------------------------------------------------------------------------

# Projects

Keep workflow identical.

Improve only visuals.

Cards should match the Board design language.

------------------------------------------------------------------------

# Calendar

Use the same glass surfaces.

Improve:

-   event cards
-   sidebar
-   toolbar
-   calendar grid
-   hover
-   selected states

------------------------------------------------------------------------

# AI Studio

AI Studio should become a premium analytics dashboard.

Redesign:

-   metric cards
-   charts
-   budget cards
-   logs
-   empty states
-   analytics sections

Charts should use:

-   elegant grid
-   muted colors
-   premium legends
-   rounded visuals
-   subtle gradients

Avoid default Chart.js appearance.

------------------------------------------------------------------------

# AI History

Use the same analytics language.

History should feel like a premium timeline.

------------------------------------------------------------------------

# Team

Improve:

-   avatars
-   member cards
-   permissions
-   invite dialog

------------------------------------------------------------------------

# Settings

Premium forms.

Glass sections.

Elegant switches.

Consistent spacing.

------------------------------------------------------------------------

# Empty States

Every empty state must be intentionally designed.

Use:

-   premium typography
-   subtle abstract illustration
-   helpful explanation
-   calm gradients

Never leave blank containers.

------------------------------------------------------------------------

# Tables

Avoid default admin tables.

Improve:

-   row spacing
-   hover
-   borders
-   typography
-   padding

------------------------------------------------------------------------

# Metric Cards

Every metric card should share one reusable component.

Contains:

-   icon
-   title
-   large number
-   optional trend
-   subtle glow

------------------------------------------------------------------------

# Dark Mode

Deep charcoal backgrounds.

Muted emerald accents.

Soft shadows.

No neon.

No cyberpunk.

------------------------------------------------------------------------

# Light Mode

Warm whites.

Soft grays.

Elegant borders.

Premium paper feeling.

Not simply inverted colors.

------------------------------------------------------------------------

# Accessibility

Maintain WCAG AA contrast.

Keyboard navigation unchanged.

Visible focus states.

------------------------------------------------------------------------

# Technical Expectations

-   Reuse components.
-   Avoid duplicated styling.
-   Create reusable design tokens.
-   Use CSS variables.
-   Keep code clean.
-   Preserve responsiveness.
